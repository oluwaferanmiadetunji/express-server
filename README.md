# express-server-generator

> A simple package to set up a mongoDb connected express server

This module

## Install

```
$ npm i -g express-server-generator
```

## Usage

In your projects folder, run

```
$ npx setup-server

Project Name (my-app):

Port number (5000):

Project Author:

Project Description:

Mongo DB connection string:

Proceed? (Y/n)

$ cd my-app

$ npm install

$ npm run start
```
